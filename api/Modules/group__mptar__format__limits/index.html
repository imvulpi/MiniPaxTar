USTAR & PAX Field Capacities, Bounds, and Buffer Limits

Fixed array sizes, maximum string lengths, encoding numeric limits, and formatting buffer sizes.

MPTAR_BINARY_MAX_8B

MPTAR_PAX_NSEC_MAX_DIGITS

MPTAR_PAX_NSEC_INITIAL_MULTIPLIER

MPTAR_PAX_MAX_DATA_LEN
Maximum payload byte capacity inside a single PAX record.
MPTAR_UINT64_STR_BUF_SIZE
Buffer capacity used to safely format a 64-bit uint string (including terminator).
MPTAR_TIMESPEC_STR_BUF_SIZE
Buffer capacity used to safely format full PAX timestamp string ("sec.nsec", including null terminator).
MPTAR_MAX_PAX_SIZE
Maximum permitted byte size for a single PAX extended header payload.
