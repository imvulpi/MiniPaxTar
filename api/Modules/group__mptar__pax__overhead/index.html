PAX Fixed Keywords & Static Overhead

Constants defining raw keyword sizes and structural character overhead for PAX records.

Defines

Name
MPTAR_PAX_KEY_LEN_PATH
Length of "path" keyword string.
MPTAR_PAX_KEY_LEN_SIZE
Length of "size" keyword string.
MPTAR_PAX_KEY_LEN_LINK
Length of "linkpath" keyword string.
MPTAR_PAX_KEY_LEN_TIME
Length of "mtime", "ctime", "atime" keywords string.
MPTAR_PAX_KEY_LEN_UID
Length of "uid" keyword string.
MPTAR_PAX_KEY_LEN_GID
Length of "gid" keyword string.
MPTAR_PAX_KEY_LEN_UNAME
Length of "uname" keyword string.
MPTAR_PAX_KEY_LEN_GNAME
Length of "gname" keyword string.
MPTAR_PAX_STATIC_CHARS
Fixed character count per record: space (' '), equals ('='), and newline (' ').
MPTAR_PAX_OVERHEAD_PATH
Total key+formatting overhead for "path" record (7 bytes).
MPTAR_PAX_OVERHEAD_SIZE
Total key+formatting overhead for "size" record (7 bytes).
MPTAR_PAX_OVERHEAD_LINK
Total key+formatting overhead for "linkpath" record (11 bytes).
MPTAR_PAX_OVERHEAD_TIME
Total key+formatting overhead for "mtime" record (8 bytes).
MPTAR_PAX_OVERHEAD_UID
Total key+formatting overhead for "uid" record (6 bytes).
MPTAR_PAX_OVERHEAD_GID
Total key+formatting overhead for "gid" record (6 bytes).
MPTAR_PAX_OVERHEAD_UNAME
Total key+formatting overhead for "uname" record (8 bytes).
MPTAR_PAX_OVERHEAD_GNAME
Total key+formatting overhead for "gname" record (8 bytes).
MPTAR_PAX_NSEC_OVERHEAD
Additional bytes for sub-second formatting: '.
