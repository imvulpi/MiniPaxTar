Archive Format Misc & Block Layout Constants

Fundamental block sizes, binary encoding flags, checksum field offsets, and scaling factors.

Defines

Name
MPTAR_OCTAL_BINARY_FLAG
High bit flag set in octal fields to indicate base-256 binary encoding.
MPTAR_USTAR_HEADER_SIZE
Byte size of a standard USTAR header block.
MPTAR_BLOCK_SIZE
Standard block layout size for tar archives.
MPTAR_SPACE_CHECKSUM_VAL
Pre-calculated checksum baseline for an empty checksum field (8 ASCII spaces).
MPTAR_CHECKSUM_OFFSET
Byte offset of the checksum field within the standard 512-byte header.
MPTAR_CHECKSUM_LEN
Byte length of the checksum field inside the header.
MPTAR_CHECKSUM_END_OFFSET
End offset of the checksum field (156).
MPTAR_EOA_BLOCK_COUNT
Number of trailing zero-filled blocks designating End-of-Archive (EOA).
MPTAR_EOA_MARKER_SIZE
Total marker size for archive termination (1024 bytes).
